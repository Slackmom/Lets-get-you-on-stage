# 📊 Presentations

This folder contains slide decks and presentation materials from the **"Lets get you on Stage – Women Edition"** sessions.

## How to Download

- **Single file:** Click on the file name, then click the **Download** button (or the ⬇ icon) on the top right.
- **All files:** Use the green **Code** button on the main repository page and select **Download ZIP** to get everything at once.

## What Goes Here

- Session slide decks (`.pptx`, `.pdf`, `.key`)
- Speaker presentations
- Keynote materials

## Naming Convention

Please name files using the format:

```
YYYY-MM-DD_Speaker-Name_Topic-Title.pdf
```

Example: `2026-03-06_Jane-Doe_Finding-Your-Voice.pdf`
