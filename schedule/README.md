# 🗓️ Schedule

This folder contains event schedules, agendas, and programme documents for the **"Lets get you on Stage – Women Edition"** sessions.

## How to Download

- **Single file:** Click on the file name, then click the **Download** button (or the ⬇ icon) on the top right.
- **All files:** Use the green **Code** button on the main repository page and select **Download ZIP** to get everything at once.

## What Goes Here

- Event agenda / programme (`.pdf`)
- Session timetables
- Speaker lineup documents
- Venue information sheets

## Naming Convention

Please name files using the format:

```
YYYY-MM-DD_Event-Name_Agenda.pdf
```

Example: `2026-03-06_Lets-Get-You-On-Stage_Agenda.pdf`
