# 📝 Templates

This folder contains ready-to-use templates for speakers and participants of the **"Lets get you on Stage – Women Edition"** sessions.

## How to Download

- **Single file:** Click on the file name, then click the **Download** button (or the ⬇ icon) on the top right.
- **All files:** Use the green **Code** button on the main repository page and select **Download ZIP** to get everything at once.

## What Goes Here

- Speaker bio templates
- Slide deck templates
- Session proposal forms
- Talk outline / structure worksheets
- Speaker introduction templates

## Naming Convention

Please name files using the format:

```
Template_Description.docx
```

Example: `Template_Speaker-Bio.docx`
