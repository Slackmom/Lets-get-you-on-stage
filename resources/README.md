# 📚 Resources

This folder contains additional reading materials, reference documents, and useful links for the **"Lets get you on Stage – Women Edition"** sessions.

## How to Download

- **Single file:** Click on the file name, then click the **Download** button (or the ⬇ icon) on the top right.
- **All files:** Use the green **Code** button on the main repository page and select **Download ZIP** to get everything at once.

## What Goes Here

- Articles and blog posts (`.pdf`)
- Book recommendations and reading lists
- External links and reference sheets
- Worksheets and handouts
- Recorded session links (as a Markdown file with URLs)

## Naming Convention

Please name files using the format:

```
Topic-Title_Resource-Description.pdf
```

Example: `Public-Speaking_Breathing-Techniques-Guide.pdf`
