# 🎤 Lets get you on Stage – Women Edition

Welcome! This repository is a document-sharing hub for the **"Lets get you on Stage – Women Edition"** sessions. Here you can find and download all materials related to the event.

---

## 📥 How to Download Files

### Download a single file
1. Click on the folder you need (see navigation below).
2. Click on the file name.
3. Click the **Download** button (⬇) in the top-right corner of the file preview.

### Download everything at once
1. Click the green **Code** button at the top of this page.
2. Select **Download ZIP**.
3. Extract the ZIP on your computer — all materials will be inside.

---

## 📁 Folder Structure

| Folder | Contents |
|---|---|
| [📊 presentations/](./presentations/) | Slide decks and presentation materials from all sessions |
| [📚 resources/](./resources/) | Additional reading materials, handouts, and reference documents |
| [📝 templates/](./templates/) | Ready-to-use templates for speakers and participants |
| [🗓️ schedule/](./schedule/) | Event agendas, timetables, and programme documents |

---

## 🤝 Contributing / Adding Files

If you are a speaker or organiser and would like to add your materials:

1. **Via GitHub:** Open the relevant folder and use the **Add file → Upload files** button.
2. **Via email/Slack:** Send your files to the repository maintainer and they will upload them for you.

Please follow the naming conventions described in each folder's README so files stay easy to find.
